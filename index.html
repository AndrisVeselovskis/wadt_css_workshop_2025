What is Sass? Your guide to this top CSS preprocessor

By Tammy Coron â€“ January 10, 2020

Discover what Sass is, and learn how to start using it.

Sass is a CSS preprocessor that adds features such as variables, nested rules and mixins into regular CSS, making coding simpler and more efficient.

What is a CSS preprocessor?

A CSS preprocessor extends CSS by allowing developers to write code in one language and compile it into CSS. Sass is the most popular one right now.

Variables example:

$primary-color: #333;
body { color: $primary-color; }

Nesting example:

nav {
  ul { list-style: none; }
  li { display: inline-block; }
}
